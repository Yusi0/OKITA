# 🎬 OKITA Canvas (v0.1.3)

> **Tauri v2 + React 19 + Rust + FFmpeg** 기반의 초고속, 무손실 모던 미디어 플레이어 & 미디어 캔버스 편집기

OKITA Canvas는 아름다운 글래스모피즘(Glassmorphism) UI와 FFmpeg 기반의 강력한 미디어 엔진을 결합하여, 동영상 및 이미지 감상부터 구간 자르기, 화면 크롭, 배속 편집, 음소거 저장, 프레임 캡처까지 빠르고 직관적으로 처리할 수 있는 데스크톱 미디어 앱입니다.

---

## ✨ 핵심 기능 (Features)

### 🎥 1. 미디어 플레이어 & 파일 탐색
- **드래그 앤 드롭 지원**: 파일을 앱 창으로 끌어다 놓아 즉시 감상 및 편집 시작.
- **스마트 탐색 키**: 키보드 방향키(`←`/`→`)로 5초 단위 이동, 스페이스바로 재생/일시정지.
- **이웃 파일 스마트 플레이리스트**: 열린 미디어가 속한 폴더 내의 다른 비디오/이미지 파일들을 자동 인식하여 연속 감상 지원.

### ⏱️ 2. 타임라인 마우스 호버 썸네일 미리보기
- 재생바 위에 마우스 커서를 올리면 해당 시점의 **실시간 썸네일 프레임**과 **정밀 타임코드(`mm:ss`)**가 마우스 커서 정중앙 바로 위에 글래스모피즘 툴팁으로 떠오릅니다.

### ⚡ 3. 원클릭 순환 배속 조절 (`0.5x` ~ `2.0x`)
- 컨트롤바 우측 배속 버튼을 클릭할 때마다 `0.5x` ➔ `0.75x` ➔ `1.0x` ➔ `1.25x` ➔ `1.5x` ➔ `2.0x` 로 즉시 배속이 순환 변경됩니다.
- 감상 시 배속은 물론, **편집 모드 미리보기 배속**으로도 바로 반영됩니다. (시청 배속은 저장 결과물에 영향을 주지 않음).

### ✂️ 4. 정밀 자르기 (Trim) & 종료 구간 자동 정지
- 원하는 비디오 구간 시작점과 종료점을 마우스 드래그로 손쉽게 지정.
- **종료 구간 자동 정지**: 미리보기 재생 중 자른 종료 지점(`trimEnd`)에 도착하면 즉시 정지되며, 다시 재생 버튼을 누르면 자른 시작 시점(`trimStart`)으로 자동 되감겨 재재생됩니다.

### 🖼️ 5. 스마트 화면 크롭 (Crop Overlay)
- 영상/이미지 화면에서 원하는 부분만 영역 지정 (`자유 비율`, `1:1`, `16:9`, `4:3`).
- 창 크기 변화 및 CSS 레이아웃 전환 중에도 영상 영역을 정확하게 추적하는 `ResizeObserver` 좌표 보정 엔진 탑재.

### 🔇 6. 편집 바 전용 음소거 편집
- 편집 모드 진입 시 슬라이더 대신 **아이콘 전용 음소거 토글 버튼** 노출.
- 편집 시 음소거 버튼을 활성화하면 백엔드 FFmpeg `-an` 옵션이 적용되어 **오디오 트랙이 완전히 제거된 무음 영상으로 저장**.
- 음소거를 켜지 않는 한 무조건 100% 원본 사운드 크기로 안전하게 저장됩니다.

### 💾 7. 스마트 비디오/이미지 저장 모달 (`ExportModal`)
- **`원본 포맷 유지`**: 재인코딩이 없는 초고속 무손실 스트림 복제 저장 (`copy`).
- **`저장 옵션`**:
  - **CRF 화질 4단계 슬라이더**: `초고화질 (CRF 18)` ~ `최소 용량 (CRF 32)`
  - **프레임레이트(FPS) 5단계 슬라이더**: `15 FPS` ~ `원본 FPS`
  - **실시간 예상 용량 이중 표기**: `원본 포맷 유지` 선택 시 용량과 `저장 옵션` 인코딩 적용 시 용량을 동시 표시하여 손쉽게 비교.

### 📸 8. 초고화질 프레임 캡처
- 동영상 감상/편집 중 원하는 순간을 원본 해상도 PNG 이미지 파일로 1초 만에 저장.

### 🔄 9. 자동 최신 버전 업데이트 알림
- 앱 시작 시 최신 버전 수신 여부를 확인하여, 새 버전이 출시되면 업데이트 모달과 함께 다운로드 링크를 제공합니다.

## ⌨️ 단축키 안내 (Shortcuts)

| 단축키 | 기능 |
| :--- | :--- |
| **`Space`** | 재생 / 일시정지 |
| **`←` / `→`** | 5초 뒤로 / 5초 앞으로 이동 |
| **`↑` / `↓`** | 볼륨 5% 증가 / 감소 |
| **`F`** | 전체화면 토글 / 해제 |
| **`M`** | 감상 소리 음소거 토글 |
| **`Esc`** | 전체화면 해제 |

## 🛠️ 개발 및 빌드 환경 (Development & Build)

### 1. 사전 요구 사항
- [Node.js](https://nodejs.org/) (v18+)
- [Rust](https://www.rust-lang.org/) (latest stable)
- [FFmpeg](https://ffmpeg.org/) (시스템 PATH에 등록되어 있거나 로컬 바이너리 보유)

### 2. 의존성 설치
```bash
npm install
```

### 3. 개발 서버 실행
```bash
npm run tauri dev
```

### 4. 프로덕션 앱 빌드
```bash
npm run tauri build
```
